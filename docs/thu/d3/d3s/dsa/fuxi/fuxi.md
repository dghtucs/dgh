# 二、逐章「考试向」复习指南（按得分性价比排序）

## ⭐⭐⭐⭐⭐ 第 14 章：排序（全卷核心）

> **这章=大头分数来源**

### 必会算法（全部会被考）

* 快速排序（重点中的重点）
* 选择（QuickSelect / LinearSelect）
* 希尔排序
* 比较次数分析

### 必会内容

* 分区思想（pivot）
* 递归深度
* 最坏 / 平均时间复杂度
* 稳定性（是否稳定）
* 原地性（是否额外空间）

📌 **典型题型**

* 手推一轮快速排序
* 比较不同排序的性能
* 问「为什么快排平均快」

👉 **建议复习占比：25% 时间**

---

## ⭐⭐⭐⭐ 第 6 章：二叉搜索树（BST）

### 必会

* BST 的定义
* 查找 / 插入 / 删除
* 中序遍历 = 有序序列
* 树高与性能

### 常考题

* 给插入序列，画 BST
* 删除某节点后的结构
* 查找复杂度分析（退化 vs 理想）

📌 **一句话记忆**

> BST 的一切性能，都取决于「树高」

# 三、你该怎么「用参考页」复习（关键技巧）

你会拿到 **20 页 PPT 参考页**，意味着：

❌ **不需要背**
✅ **必须能“看到就会用”**

### 所以你复习时要做到：

* 看到公式 → 知道什么时候套
* 看到算法步骤 → 知道下一步干什么
* 看到定义 → 知道对应哪类题

👉 **千万别只看 PPT，一定要“做题式复习”**
