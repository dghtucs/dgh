- J - 刚度矩阵 (Stiffness Matrix)
- K - 下三角矩阵 (Lower Triangular Matrix)
- L - 马尔可夫矩阵 (Markov Matrix)
- M - 零空间矩阵 (Nullspace Matrix)

- O - 投影矩阵 (Projection Matrix)

- Q - 上三角矩阵 (Upper Triangular Matrix)
